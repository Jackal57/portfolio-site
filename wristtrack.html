WristTrack

This project is a software program written in Java that is designed to help prevent carpal tunnel. I created this project for my Senior Capstone assignment in my final year of undergrad. The program has 3 main components:
  • An input bar that raises each time a user performs physical actions at their computer
  • A screen that displays once the bar fills up completely, allowing the user to configure the sensitivity of the program
  • A screen that instructs the user to take a break and sets a timer indicating when they should resume
